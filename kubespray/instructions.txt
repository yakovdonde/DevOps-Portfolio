Clone Kubespray git repo:
    git clone https://github.com/kubernetes-sigs/kubespray.git

Switch to kubespray directory:
    cd kubespray

Copy the sample inventory directory to the directory with you cluster
    cp -rf inventory/sample inventory/donde-k8s-cluster

Adjust the default inventory file with your data:
    vi inventory/donde-k8s-cluster/inventory.ini

Create inventory file:
    touch donde-k8s-inventory.txt

Fill the donde-k8s-inventory.txt with the contents from donde-k8s-inventory.txt in this repo

Run Kubespray with this inventory file:
    ansible-playbook cluster.yml  -i donde-k8s-inventory.txt --become --become-user=root
